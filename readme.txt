Add beautiful Web 2.0, XHTML valid videobox anywhere on your site. The plugin is easily customizable through widget panel. This plugin is ideal for music, tutorial, ads videos. Check for more on <a href="http://tomsramek.com/plugins/advanced-videobox/">the plugin's page</a>
== Description ==

With this plugin you can add videos to your sidebar (or any other widgetized of your site). Just copy and paste code of the video into the Advanced widget.
The Youtube’s original video code doesn’t pass W3C validation. The filtered video code of the Asvanced Videobox will keep your site XHTML valid. Check for more on <a href="http://tomsramek.com/plugins/advanced-videobox/">the plugin's page</a>

* Easy to manage
* Option to change width and height
* Option to disable parameters that produce invalid XHTML code
* Web 2.0 graphics
* Add up to 20 videos
* Easily customizable via CSS
* Ideal plugin for music, tutorials, ads videos

== Installation ==
1. Upload `advanced-videobox` to the `/wp-content/plugins/` directory.
2. Activate `Youtube VideoBox Plugin` through the 'Plugins' menu in WordPress.
3. Check for more on <a href="http://tomsramek.com/plugins/advanced-videobox/">the plugin's page</a>
